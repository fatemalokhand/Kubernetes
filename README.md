## Overview
This project contains a Kubernetes setup for a
multi-service application deployed on AWS EKS/AKS using GitOps principles.

## Key Services
- Kubernetes Deployments, Services, Ingress
- Horizontal Pod Autoscaling
- Helm chart 
- GitOps CD with Argo CD
- CI pipelines
- Monitoring with Prometheus and Grafana

## Technologies
Kubernetes, Docker, Helm, Argo CD, Prometheus, Grafana, GitLab CI, AWS EKS
